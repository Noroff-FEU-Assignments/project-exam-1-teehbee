# Project Exam 1

<img src=assets/project-portfolio.jpg>

Project exam for the first year of front-end development studies at Noroff school og technology and digital media.

## Description

The task of this project exam was to create a blog page with posts being fetched from the Wordpress API. I chose to go with a tourism blog for my region of Valdres in Norway.

- Wordpress was used to create blog posts which were appended to the website

- The page was to contain a carousel and a blog list with a "load more" function for blog posts.

- The page also had to contain a contact form with JS form validation.

## Built with

- HTML

- CSS

- Vanilla JS
